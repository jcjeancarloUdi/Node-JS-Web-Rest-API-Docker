Docker para desenvolvedores

Criar e implantar novos aplicativos é mais rápido com contêineres. Os contêineres do Docker encerram o software e suas dependências em uma unidade padronizada para desenvolvimento de software que inclui tudo o que ele precisa para executar: código, tempo de execução, ferramentas do sistema e bibliotecas. Isso garante que seu aplicativo sempre será executado da mesma forma e torna a colaboração tão simples quanto o compartilhamento de uma imagem de contêiner.

Compose é uma ferramenta para definir e executar aplicativos Docker com vários contêineres. Com Compose, você usa um arquivo YAML para configurar os serviços do seu aplicativo. Então, com um único comando, você cria e inicia todos os serviços da sua configuração.
